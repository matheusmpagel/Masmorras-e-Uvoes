# Masmorras e Uvões

Jogo de Ação 2D Top-Down, em que o jogador usa um guerreiro uva para avançar por fases diferentes, ficando mais forte com o passar do tempo. 
Possui um combate à distância, com foco em derrotar monstros.
Consiste em 3 fases, cada uma com uma quantidade fixa de monstros. Cada tipo de monstro possui uma quantidade de vida, de dano e de velocidade diferentes.

**CONTROLES**
Movimento - WASD
Ataque - M1 / LMB / Botão esquerdo do mouse
Mira - Mouse
Pausar - Esc

**MECÂNICA DE LOOT**
Inimigos podem dropar itens ao serem derrotados. 
Coração = Recupera 1 vida do jogador
Estrela = Aumenta permanentemente o dano do jogador (É mais rara que o coração)

*Jogo produzido durante as aulas de Tópicos Especiais em Tecnologia do professor Stefan Lucas, com o tema "Uvas e Armas"*
